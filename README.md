# homebrew-tap

Personal Homebrew tap for [@MaVericKWareZ](https://github.com/MaVericKWareZ).

## Available formulae

| Formula | Description |
| --- | --- |
| [chime](https://github.com/MaVericKWareZ/chime) | Friendly terminal alarms, timers & pomodoro |

## Usage

```bash
brew tap MaVericKWareZ/tap
brew install chime
```

Or in one shot:

```bash
brew install MaVericKWareZ/tap/chime
```

Upgrade with `brew upgrade chime`.
